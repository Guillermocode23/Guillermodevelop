Lenguajes y Herramientas

Mis proyectos están diseñados para resolver problemas reales que enfrentan los negocios modernos, especialmente en la gestión eficiente de inventarios, productos y proveedores.

Proyecto crud_java: Un sistema con Java y MySQL para administrar productos y proveedores, ideal para tiendas o negocios que necesitan gestionar su stock y catálogo.

Proyecto crud-spring: Una aplicación basada en Spring Boot que expone una API REST, facilitando la integración con sitios web y aplicaciones modernas, permitiendo gestionar productos desde el navegador y controlar inventarios de forma dinámica.

Estas soluciones permiten construir sistemas de catálogo en línea, paneles administrativos y dashboards con estadísticas que impulsan la digitalización y eficiencia en negocios de cualquier tamaño.
